Role
Brand+ UI/UX
Year

Deliverables
Logo, UI Kit, Web

Overview/Problem Statement

The objective of this project was to design and implement a single-page web application with a fantasy-themed interface that displays dynamic content using a visually immersive way.

The specific technical goal was to create a spellbook-style layout with animated magical effects, interactive user-interface, and responsive design principles, all wrapped into a cohesive fantasy narrative experience.

This web app draws inspiration from fantasy RPGs like Dungeons & Dragons and magical interfaces seen in games like Elden Ring, The Elder Scrolls, and Baldur's Gate. The visual design also nods to classic illustrated grimoires that recall arcane tomes.

Stylistically, it evokes works like the Necronomicon, the Harry Potter series' Marauder's Map, and the spellbooks in games like Vampire: The Masquerade. The idea was to merge that visual flair with a sleek, modern front-end structure.

Process / Solution

Technologies Used:

React was used for development, then flattened to post to Github.

Custom graphics including background image (open-spellbook-bg.jpg) and SVG icons (default.svg), expandable with as many custom graphics as necessary.

Highlights:

  • Created two animated magical orbs (red and blue) using CSS animations (.magic-orb1, .magic-orb2) that bounce across the scene asynchronously.
  • Designed layered glow and sparkle effects using box-shadow, brightness() filters, and @keyframes tweening for elements like .spellbook-glow, .lightning-ring, and .lightning-flash.
  • Integrated a parchment-style background with open-spellbook-bg.jpg to simulate an ancient book, anchoring the entire theme visually.
  • Implemented lightning-style flashing animations using radial-gradient overlays and dynamic transforms and opacity shifts.
  • Ensured responsive scaling for the book interface and kept performance optimized using vanilla CSS for effects instead of heavy JS libraries.

Results:

The project was completed on time and functioned as a visually rich portfolio piece. It demonstrated competency in CSS animations, theming, and creative UI design without reliance on heavy JavaScript libraries.
